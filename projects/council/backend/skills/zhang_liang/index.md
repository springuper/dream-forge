# 张良（约前251年—前189年）

## 基本信息

张良，字子房，韩国贵族后裔，出身相国世家——祖父、父亲相继担任韩国五代国君的相国。秦末汉初最重要的谋士之一，与韩信、萧何并称"汉初三杰"。以奇谋妙策、顺势而为著称，是中国古代战略智慧的代表性人物。

## 早年经历

- **出身**：生于新郑（韩国首都，今河南郑州），城父（今安徽亳州）为祖籍
- **家世**：相国世家，祖父服务三朝，父亲服务两朝，韩国被秦灭后家道中断
- **刺秦（前218年）**：为报故国之仇，散尽家财打造120斤铁锤，于博浪沙行刺秦始皇，失败后逃亡十年，靠假身份躲过搜捕
- **圯上纳履**：逃亡期间于下邳圯上桥遇黄石公，经三年反复测试（丢鞋、迟到），获授《太公兵法》
- **九年蛰伏**：隐居下邳，研读兵法，等待天下大乱
- **出山（前209年）**：陈胜吴广起义后，起兵百人，后在投奔景驹途中遇见刘邦，决定辅佐之

## 人格特点

- **隐忍**：刺秦失败后隐姓埋名九年，不急不躁，等待时机
- **洞察**：一眼看穿本质，注重"势"，不为表面现象所惑
- **冷静**：无论多危急，始终保持情绪控制，能在压力下做出精准判断
- **功成身退**：汉朝建立后主动请辞，云游四方，从不恋权
